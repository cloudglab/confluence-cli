Wiki / 检索

先做成静态检索：安装、角色、内容查询、REST 透传、发布、附件、评论和环境变量都能在本页快速过滤。后续可以扩展为构建期索引，覆盖 README、CHANGELOG 和源码导出的命令清单。

输入关键词后过滤下方命令。

常用场景

  • 搜索页面confluence searchContent --cql 'space = "DEV" AND text ~ "API"' --limit 10
  • 查看页面详情confluence getContent --id 123456
  • 发布 Markdownconfluence uploadMarkdown --file docs/page.md --space DEV --toc --confirm true
  • 下载页面和附件confluence downloadPage --id 123456 --outputDir exports --downloadAttachments true
  • 查看 REST 端点confluence listRestApis --group content --limit 20
  • 透传 REST 查询confluence callRestApi --method GET --path '/content/{id}' --pathParams '{"id":"123456"}'

基础 / 安装

  • 一键安装npx -y @cloudglab/confluence-cli@latest install
  • 更新 CLIconfluence update
  • 查看版本confluence version
  • 列出命令confluence list
  • 查看参数confluence help uploadMarkdown
  • 初始化配置confluence initConfluence --url https://confluence.example.com --pat token

角色入口

  • 完整能力confluence --role full list
  • 只读入口confluence-reader list
  • 写入入口confluence-writer list
  • 内联角色confluence --role=reader searchContent --limit 10

内容 / 评论

  • 页面详情confluence getContent --id 123456
  • 子页面confluence getPageChildren --id 123456 --type page
  • 查评论confluence getComments --id 123456
  • 加评论confluence addComment --id 123456 --text "已更新" --confirm true
  • 删除页面confluence deleteContent --id 123456 --confirm true

标签 / 附件

  • 查看标签confluence getLabels --id 123456
  • 添加标签confluence addLabels --id 123456 --labels '["api","docs"]' --confirm true
  • 列出附件confluence listAttachments --id 123456
  • 上传附件confluence uploadAttachment --id 123456 --file docs/a.png --confirm true
  • 下载附件confluence downloadAttachment --id 123456 --attachmentId att123

发布 / 转换

  • Markdown 转 wikiconfluence convertMarkdownToWiki --file docs/page.md
  • 生成 mark metadataconfluence generateMarkMetadata --file docs/page.md --space DEV
  • 上传 Markdownconfluence uploadMarkdown --file docs/page.md --space DEV --toc --confirm true
  • 上传 HTMLconfluence uploadHtml --file dist/page.html --space DEV --confirm true

REST / 调试

  • 列出 content 端点confluence listRestApis --group content --limit 20
  • 只看 GETconfluence listRestApis --method GET --limit 20
  • 调用 GET 端点confluence callRestApi --method GET --path '/content/{id}' --pathParams '{"id":"123456"}'
  • 调用写端点confluence callRestApi --method POST --path '/content' --body '{"type":"page"}' --confirm true

工程命令

  • 类型检查pnpm typecheck
  • 运行测试pnpm test
  • 构建产物pnpm build
  • 发版前 smokepnpm release:smoke-query --dry-run

环境变量

  • 服务地址export CONFLUENCE_URL="https://confluence.example.com"
  • PAT 凭证export CONFLUENCE_PAT="your-personal-access-token"
  • 自定义 API 地址export CONFLUENCE_API_BASE_URL="https://confluence.example.com/rest/api"
  • 禁用真实写入export CONFLUENCE_DISABLE_WRITE=true
  • 关闭更新探针export CONFLUENCE_SKIP_UPDATE_CHECK=true
写操作默认受保护；真实写入仍需显式传入 confirm=true，也可以用 CONFLUENCE_DISABLE_WRITE=true 全局禁用。
